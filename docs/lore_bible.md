# FrontendLegends 设定集

## 世界与门派

FrontendLegends 是一个以现代前端技术为武学隐喻的江湖。角色将编码、思维、架构、招架与灵感视作五类基础功法；不同门派以各自的技术体系将其发展为高级功法。

- NG 神教：以模板、组件、依赖注入与路由守卫为核心。
- 香草派：擅长 DOM、事件与原生 Web 技法。
- 量子仙宗：以量子框架与响应式技法修行。
- 鱿鱼山庄：以 JavaScript 与异步流程见长。

地图、NPC 和物品的拼音 ID 是世界内容的稳定标识，不是代码术语；TMX 地图会直接引用它们，因此不随运行时字段更名。

## 角色成长

角色拥有四项属性：编码（strength）、思维（agility）、架构（constitution）与灵感（wisdom）。基础功法面向所有角色，高级功法则要求加入对应门派并装备在相应槽位。

`cultivation` 是修为的唯一运行时和存档字段名。冥想先积累当前精力 `mp`；精力达到由装备架构功法和架构属性决定的上限时，会转换为 1 级 `cultivation` 并清空当前精力。每级 `cultivation` 提供 2 点当前精力上限，并间接提高体力上限。

Token（`money`）与潜能（`potential`）均为从零开始的成长资源。Token 用于交易，潜能用于学习和训练。

## 数据与存档

当前发行版本为 0.7.5。存档格式为 v2，文件名为 `frontend_legends_save_v2.json`。v1 存档字段已被移除；旧存档不会加载，也不会迁移到新格式。
